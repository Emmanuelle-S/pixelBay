Exo 2

Énoncé :

Pour améliorer la gestion du catalogue de jeux de PixelBay, vous devez créer et manipuler un tableaux en JavaScript. Votre tâche consiste à créer un tableau contenant une liste de noms de jeux vidéo, puis à effectuer quelques opérations simples sur ce tableau.

Votre programme doit accomplir les tâches suivantes :

Création du Tableau de Jeux : Créez un tableau jeux contenant au moins cinq noms de jeux vidéo.

Affichage du Deuxième Jeu : Affichez le nom du deuxième jeu du tableau dans la console.

Modification du Troisième Jeu : Modifiez le nom du troisième jeu du tableau, puis affichez le nouveau nom dans la console.
