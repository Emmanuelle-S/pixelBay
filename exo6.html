Exo 6

Objectif :

Utilisation de la manipulation du DOM et compréhension des tableaux et de la structure switch case en JavaScript.

Énoncé :

Chez PixelBay, chaque jeu vidéo a un code numérique associé (1,2,3,4,…). En tant que gérant de la boutique, vous devez afficher le titre du jeu dans le conteneur HTML prévu à cet effet en fonction du numéro entré par l'utilisateur.

Voici la correspondance entre les numéros et les jeux :

  • "Space Adventure"
  • "Castle Quest"
  • "Mystery Island"
  • "Robot Wars"

Le code HTML, CSS et JavaScript de départ est fourni. Vous devez compléter le code JavaScript pour que, lorsque l'utilisateur entre un numéro (1, 2, 3 ou 4) via prompt(), le titre correspondant du jeu s'affiche dans le conteneur.
