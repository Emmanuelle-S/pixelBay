Exo 18

Énoncé :

Votre nouvelle mission est de construire un programme dynamique qui détermine quel jeu parmi deux titres phares a conquis le cœur des gamers à travers le monde.

Concevez deux objets représentant deux jeux vidéo légendaires. Chaque objet doit inclure :

  • Titre : Le nom évocateur du jeu.
  • Année de Sortie : L'année où le jeu a fait ses débuts.
  • Genre : Le style du jeu, tel que Arcade, Aventure, etc.
  • Score : La note globale du jeu sur 100, reflétant son succès.

Chacun de ces objets doit posséder deux fonctions essentielles :

  • affichage() : Un appel de projecteur sur le jeu, affichant toutes ses informations clés.
  • getPopularite() : Un calcul stratégique qui détermine la popularité du jeu en utilisant la formule : Popularité = (score * année de sortie) / 100.

Défis :

  • Affichage Stratégique : Mettez en scène un duel captivant entre les deux jeux. Qui remportera le titre du jeu le plus populaire ?
  • Calcul de la Popularité : Utilisez vos compétences en programmation pour calculer et afficher la popularité de chaque jeu.
